# Dawiyat Executive Dashboard

This Streamlit dashboard reads data directly from `Dawiyat Master Sheet.xlsx` stored in the same GitHub repository.

## Files required in the repo
- `app.py`
- `requirements.txt`
- `README.md`
- `Dawiyat Master Sheet.xlsx`

## Deploy on Streamlit
1. Push all files to GitHub in the same folder.
2. In Streamlit Community Cloud, point the app to `app.py`.
3. Keep Python at 3.12.
4. Reboot the app after any workbook replacement.

## Notes
- No web upload is required in this version.
- Filters are driven from the workbook saved in GitHub.
- Penalties and PMO summary are included.
